# Topology-Augmented Synthetic Control Results Package

This repository contains the organized results package for a synthetic-control analysis with topological data analysis (TDA) diagnostics and topology-augmented robustness checks.

The package is designed to make the empirical results auditable. It collects existing outputs from the main synthetic-control workflow, topology diagnostics, baseline placebo inference, topology-augmented SCM comparison, and leave-one-out robustness checks.

This repository does **not** re-estimate models. It stores and organizes output files that were generated by previous analysis notebooks.

---

## Project Overview

The project compares conventional synthetic-control methods with topology-aware diagnostics and topology-augmented synthetic-control outputs.

The empirical workflow has three main components:

1. **Synthetic-control construction**

   Baseline SCM and author-weight SCM are used to construct synthetic counterfactual paths for the treated unit.

2. **Topological diagnostics**

   Delay embeddings and persistent homology summaries are used to evaluate whether the treated and synthetic paths share similar structural behavior, especially through H1 features.

3. **Topology-augmented SCM and robustness**

   A topology-augmented SCM specification is evaluated against the baseline SCM. Leave-one-out robustness checks test whether the topology-augmented results depend heavily on influential donor states.
